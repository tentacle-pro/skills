# Templates 文件夹 — 智能体指南

## 用途

存放**所有笔记模板**。使用模板可以快速创建结构一致的笔记，减少重复工作。

## 核心规则

- 模板文件使用描述性命名，如 `daily-note.md`、`project-readme.md`
- 模板中的占位符使用 `{{变量名}}` 格式（兼容 Obsidian Templater 插件）
- 不要在此文件夹存放实际内容笔记，只存放模板
- 新增模板类型时，在本 AGENTS.md 中登记

## 现有模板清单

| 文件名 | 用途 |
|---|---|
| `daily-note.md` | 每日笔记模板 |
| `project-readme.md` | 项目 README 模板 |
| `area-overview.md` | 领域概览模板 |
| `resource-index.md` | 资源主题索引模板 |
| `meeting-note.md` | 会议记录模板 |
| `book-note.md` | 读书笔记模板 |

## 智能体操作指引

- 创建新笔记前，先检查此文件夹是否有合适的模板
- 复制模板内容到目标位置，替换占位符后使用
- 如需新增模板，在此文件夹创建文件并更新上方清单
