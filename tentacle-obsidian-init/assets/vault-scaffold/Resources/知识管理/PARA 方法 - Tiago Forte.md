# PARA 方法：一套简单易用的系统，助您在几秒钟内整理好您的数字生活

想象一下完美的组织系统。

一个能告诉你生活中每条信息应该放在哪里的系统——每份文件、档案、笔记、日程安排、提纲和研究资料——以及在需要时能在哪里找到它。

这样的系统必须设置起来极其简单，维护起来也同样容易。毕竟，只有最简单、最轻松的习惯才能长期坚持下去。

它既要灵活，能够适应你人生不同阶段的需求，又要全面，这样你才能在存储信息的众多地方使用它，例如计算机的文件系统、云存储平台（例如 Dropbox 或 Google Drive）或数字笔记应用程序。

但最重要的是，理想的组织系统应该能够直接为你的职业生涯和生活带来切实的好处。它能显著加快你完成项目和实现目标的速度，而这些项目和目标对你来说至关重要。

换句话说，**组织生活的终极系统是*可操作的*系统。**

与其给自己设置更多障碍、推迟那些能带来改变的行动，不如把这些行动提前，让它们更容易开始和完成。

经过十多年的个人试验、数千名学生的教学以及对世界级专业人士的指导，我开发出了这样一套系统。

**它名为 PARA——一个简单、全面但又极其灵活的系统，用于组织任何平台上的任何类型的数字信息。**

我向你保证，它不仅能让你的生活井然有序，还能为你提供一套工具，让你能够巧妙地掌控信息流，从而实现你所下定决心的一切目标。

## 涵盖你人生方方面面的四大类别

PARA 基于一个简单的观察：你生活中的所有信息都属于四个类别。

你有一些正在积极进行的**项目**——一些短期工作（无论是在工作中还是生活中），你都带着特定的目标去完成它们。例如：

- 完成网页设计
- 购买新电脑
- 撰写研究报告
- 翻新浴室
- 完成西班牙语课程
- 摆放新的客厅家具

你肩负着一些**责任领域**——涵盖你工作和生活中需要持续关注的重要方面。这些方面可能包括：

- 工作职责：市场营销、人力资源、产品管理、研发、直接下属管理或工程等
- 个人责任：健康、财务、子女、写作、汽车或家庭

然后，你可以找到一系列你感兴趣并正在学习的主题的**资源**，例如：

- 平面设计
- 个人效率
- 有机园艺
- 咖啡
- 现代建筑
- 网页设计
- 日语
- 法国文学
- 记笔记
- 呼吸练习
- 习惯养成
- 摄影
- 营销资产

最后，还有**存档**，其中包含前三个类别中不再活跃但你可能希望保存以供将来参考的所有内容：

- 你已完成或已暂停的项目
- 不再活跃或相关的领域
- 你不再感兴趣的资源

就是这样！四个顶级文件夹——项目、领域、资源和档案——每个文件夹都包含少量子文件夹，分别用于存放你生活中的每个活跃项目、责任领域、资源和档案。

你或许很难相信，像你这样复杂而现代的人类生活竟然可以简化为仅仅四个类别。你或许觉得，你需要处理的事情远远超过这样一个简单的系统所能容纳的范围。

但这正是问题的关键：如果你的组织系统像你的生活一样复杂，那么维护它的需求最终会剥夺你过好生活所需的时间和精力。

你用来整理信息的系统必须足够简单，才能*解放*你的注意力，而不是占用你的更多注意力。你的系统应该为你节省时间，而不是占用你的时间。

## 关键原则——按项目和目标组织信息

我们大多数人都是在学校里第一次学会如何整理信息。老师教导我们要按*学科*分类整理课堂笔记、讲义和学习资料，例如数学、历史或化学。

但不知不觉中，我们把这种做法延续到了成年。我们仍然按照"市场营销"、"心理学"、"商业"或"创意"等极其宽泛的主题对文件和资料进行分类。

这在你毕业后的职业生涯中完全没有意义。在职场上，没有课程，没有考试，没有成绩，也没有文凭。没有老师会告诉你期末考试该写什么，因为根本没有期末考试。

无论在工作还是生活中，你所拥有的都是*你想要达成的目标*。你可能想要推出新产品、计划家庭度假、做出关键决定、在附近找到托儿所、发表新文章，或者达到季度销售目标。

在你忙碌的一天中，当你努力完成这些事情时，你*绝对没有时间*去翻阅像"心理学"这样庞大的类别，去寻找你需要的那一条信息。

我建议你不要像在学校里那样按宽泛的主题整理信息，而是根据你目前正在进行的项目和目标来整理。这就是"按行动性整理"的含义——我会在整本书中反复强调这个理念。

例如，当你坐下来开始一个平面设计项目时，你需要将所有与该项目相关的笔记、文档、素材和其他材料集中在一个地方，随时准备就绪。

这看似显而易见，但实际上却恰恰相反。大多数人倾向于把所有相关资料分散在十几个不同的地方，光是找到它们就要花掉半个小时。

如何确保与每个项目或目标相关的所有资料都集中在一个地方？首先，你要做好整理工作。这样你就能准确地知道每样东西应该放在哪里，以及在哪里可以找到它们。

## 按项目组织的力量

我曾在旧金山湾区担任效率教练数年。那正是科技繁荣的鼎盛时期，来自全球最具影响力公司的高管们都在寻求提升工作效率的任何方法。

我曾在南旧金山一家知名生物科技公司辅导几位高管，公司园区环境优美，可以俯瞰海湾。我记得在一个阳光明媚的春日，我正在等待我的下一位客户——一位负责研发几种新型救命药物的高级总监。

他到达后，我们的辅导课程像往常一样开始，我问了一个简单的问题："你有项目清单吗？"

作为一名效率教练，在与客户合作时，我首先会要求他们提供项目清单。我需要了解他们的工作内容、当前的工作量以及他们希望推进的优先事项和目标。

他说"当然！"，然后凭记忆快速列了个清单（这是第一个不祥之兆）。

你看出问题了吗？

根据我们之前的定义，这份清单上的每一项都算不上项目。"战略规划"真的有尽头吗？你真的能把"休假"从清单上彻底划掉吗？但愿不会！

这份清单上的每一项实际上都是**责任领域**。这看似咬文嚼字，实则不然。我发现，无论你多么聪明或多么有干劲，有两件至关重要的事情你都做不到，除非你把你的责任领域细化成具体的项目。

### 1. 你无法真正了解自己承诺的程度

我最常听到的抱怨之一就是人们"没有足够的精力"。我能理解——你是不是也经常感觉自己肩上的担子太重？

但只要你用领域划分的视角来看待工作，就永远无法真正了解自己肩负的工作量。看看上面的列表，"招聘"这项工作究竟占了多少比例？它可能只是每六个月招聘一名兼职员工，也可能意味着本季度要填补 50 个职位空缺。

一眼根本看不出来，这种不确定性会导致每个方面都感觉比实际情况更繁重。

想象一下，如果你把招聘部门的每个项目都列出来，每天都把这份清单放在眼前，是不是就能更容易地知道还有多少工作要做，以及下一步该做什么？

### 2. 你无法将当前努力与长期目标联系起来

知识工作最具挑战性（但也最有成就感）的方面之一，就是它需要我们的创造力。而创造力如果没有动力，就很难持续发挥。如果你精疲力竭、意志消沉，就无法继续进行最佳思考，也无法贡献最佳创意。

我们的动力取决于什么？主要取决于持续进步。如果我们知道努力终会*带来回报*，那么短期内我们就能承受相当大的压力和挫折。

这就引出了第二个问题：如果没有一份具体项目的清单，你就无法将你当前的努力与你的长期目标联系起来。

再看看上面的清单。清单上的每一项都不会结束或改变——这就是责任领域的定义，它会无限期地持续下去。现在想象一下，每周、每月、甚至每年醒来，面对的都是同样的、永无止境的责任清单，这会造成怎样的心理影响。无论你多么努力，那无尽的地平线似乎永远也无法靠近。

说实话，就算我想，也想不出比这更好的方式来扼杀你的积极性了。

当你把职责分解成一个个易于管理的小项目时，就能确保项目清单不断更新。这种更新换代会带来持续不断的成就感，让你每次成功完成一个项目都能庆祝一番。

无论你的职责范围有多广，你*都*可以将其分解成更小的项目。如果你想知道自己是否真的在朝着目标前进，就必须这样做。

## 为你想要的生活做好准备

使用 PARA 不仅仅是创建一堆文件夹来存放文件。

它关乎明确你的工作和生活结构——你致力于什么，你想改变什么，以及你想去往何方。它关乎以一种能够支持并促成你理想未来生活的方式组织信息。

我们所谓的"整理"很多时候，其实只是变相的拖延。我们告诉自己"正在准备"或"正在做调查"，假装这是在取得进展。而实际上，我们只是在寻找任何可以修饰或整理的小事，以此来逃避面对我们真正恐惧的任务。

PARA 打破了这种表象，为我们提供了一种极其简单的组织方法，让你无需任何借口，也无需再做任何事，只需专注于下一步的关键步骤。它以极简主义的方式，为你的环境增添*恰到好处的*秩序，让你拥有清晰的思路继续前进，仅此而已。

还有其他更复杂、更精细、更专业的组织信息的方法，但 PARA 是唯一经受住时间考验的方法，因为它能为你节省的时间比花费的时间更多。
