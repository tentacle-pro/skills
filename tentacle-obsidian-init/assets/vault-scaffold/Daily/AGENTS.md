# Daily 文件夹 — 智能体指南

## 用途

存放**每日笔记（Daily Notes）**。每天一个文件，记录当日的想法、任务、会议记录、随手笔记等。

## 核心规则

- 文件命名格式：`YYYY-MM-DD.md`（例如：`2026-03-19.md`）
- 每日笔记是捕捉想法的**收件箱**，内容后续应整理到 PARA 对应文件夹
- 不要在 Daily 中长期存放需要持续维护的信息
- 使用 Templates 文件夹中的 `daily-note.md` 模板创建新的日记

## 建议内容结构

每日笔记可包含以下区块（按需使用）：

```markdown
## 今日重点
- [ ] 任务1
- [ ] 任务2

## 笔记 / 随想

## 会议记录

## 待处理（Inbox）
- 收集到的信息、链接、想法，待后续整理
```

## 智能体操作指引

- 创建新的每日笔记时，使用 `Templates/daily-note.md` 作为模板
- 文件名严格使用 `YYYY-MM-DD.md` 格式
- 每日笔记中的"待处理"内容，应定期（每周）整理分发到 Projects / Areas / Resources
- 搜索特定日期的记录时，直接按文件名检索
